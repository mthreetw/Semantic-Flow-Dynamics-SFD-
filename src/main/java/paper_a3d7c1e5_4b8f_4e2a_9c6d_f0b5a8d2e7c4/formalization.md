## 起始條件：四股力量的耦合

來源：論文第四節

### 4.1–4.4 四股力量

IndustrialStandardization ≡ 十九世紀末引入的普魯士模型。設計目標：大規模生產標準化執行者
ConfucianCulture ≡ 尊師重道、服從權威、道德教化。為學校權力結構提供本土合法性
ImperialExamResidue ≡ 科舉制度 1300 年運作的殘留。考試決定命運 ∧ 背誦是核心操作 ∧ 學歷通往權力
Poverty ≡ 資源約束。背誦是資源約束下的最低成本解

### 4.5 耦合

ForceCoupling ≡ {IndustrialStandardization, ConfucianCulture, ImperialExamResidue, Poverty}

四股力量在 SignalMatching 上重疊：
  IndustrialStandardization → SignalMatching（按規格執行）
  ConfucianCulture → SignalMatching（按經典行事）
  ImperialExamResidue → SignalMatching（按範文作答）
  Poverty → SignalMatching（最便宜的操作）

ForceCoupling 結構穩定 ← 拆掉任何一股，另外三股還在撐

### 4.6 探針：邏輯學的缺席

LogicAbsence ≡ 邏輯學在中國教育系統中的缺席

邏輯學作為 40a22ff8-9d90-4e1e-82f6-1fd80917c139.Filtering 結構：
  學生具備「這個推理有效嗎」的過濾條件 → 此條件跨學科運作 → ExamCriterion 在邏輯過濾面前溶解

排異分析：
  IndustrialStandardization：¬排斥邏輯（工程師需要邏輯）
  ConfucianCulture：排斥邏輯（知識合法性繫於權威而非推理過程）
  ImperialExamResidue：排斥邏輯（判準考匹配而非推理有效性）
  S₂（政治權力）：排斥邏輯（對前提的追問 = 威脅）
  Poverty：¬直接排斥 ∧ 在其他三股排斥的條件下提供額外藉口

## 模型

來源：論文第五節

### 5.1 循環結構

主迴路：
  S₁ ∧ S₂ → Criterion → Execution → Output → Reception → Feedback → Criterion
  循環標註：主迴路
  退出條件：九條侵蝕線（見正反饋的不穩定性段）

自我複製迴路：
  Output → Execution（Output 成為教師 ∧ 家長 ∧ 決策者，回流到 Execution ∧ Criterion）
  循環標註：自我複製迴路
  退出條件：Output 停止回流（LyingFlat ∨ Emigration）

Execution 自我增強迴路：
  Output（升學率）→ Execution 權威 → 更好生源 ∧ 更多資源 ∧ 地方政府支持 → Execution 權力更大 → Execution 更強化同一套操作 → Output（更高升學率）
  循環標註：Execution 自我增強迴路
  退出條件：生源池縮小 ∨ 升學率回報消失

三條迴路同時鎖定，互相加固

### 5.2 各環節定義

S₁ ≡ 經濟需求：需要什麼能力的人
S₂ ≡ 權力需求：需要什麼程度可控的人
SocialDemand ≡ S₁ ∧ S₂

以前：S₁ ∧ S₂ ¬衝突（經濟需要標準化執行者 ∧ 權力需要服從的人 → 同一個解）
現在：S₁ ⊗ S₂（經濟需要 ModelingCapability → 獨立思考 → 威脅 S₂）

Criterion ≡ 循環的錨。所有下游環節被錨拉住。當前錨：高考（ExamCriterion）

Execution ≡ 學校執行。當前操作：背誦（ForceCoupling 下的最低成本解）

Output ≡ 循環的輸出
  40a22ff8-9d90-4e1e-82f6-1fd80917c139.EpistemicBarrier 在 Output 中代際傳遞
  Output 是模型簡化：大量人在 Execution 過程中被篩掉。被篩掉的人是循環的燃料，不是產品

Reception ≡ 社會接收。匹配（穩定）∨ 落差（毒打）

Feedback ≡ 痛感能不能傳回權力端
  痛感落在權力身上 → Feedback 通路開
  痛感落在學生身上 → Feedback 通路斷

### 5.3 穩定態

StableState 前提：Output 確實有用
從歷史推出：工業化階段 S₁ 要的是 SignalMatching → 循環輸出的恰好是這個 → S₁ ∧ S₂ 都和 Output 匹配 → 循環穩定

穩定性三重加固：
  自我複製迴路（Output 回流為操作者）
  ∧ 利益路徑正反饋（考上 → 學歷 → 工作 → 收入，回報可見，驅動投入）
  ∧ 底層主動支撐（高考是窮人唯一上升通道——理性選擇）

轉折：S₁ 從「需要標準化執行者」→「需要 ModelingCapability」→ 循環仍輸出 SignalMatching → 市場不收
