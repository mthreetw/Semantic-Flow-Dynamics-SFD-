## 本文概念定義

來源：論文 §0.2

以下概念為本文在 SFD 基礎上發展的分析工具。不修改 SFD 任何定義。

### 信模

XinModel ≡ 從長時段歷史的反覆行為模式中構建的工作模型
  XinModel 是對 Xin 的最佳近似，不是 Xin 本身
  品質由歷史回歸測試決定

### 核心信判準

CoreXinCriterion ≡ 識別核心信的操作工具
  需要 IdentityComposition ∧ DeathCriterion 同時滿足，缺一不可

IdentityComposition ≡ 從長時段歷史中提取的、區別於其他文明的獨特行為傾向
  排除人類共有特質（所有文明都需要食物、安全、秩序）
  只留無法被其他文明共享的傾向

DeathCriterion ≡ 文明是否付出最大代價（死亡）保護該傾向
  死亡是代價上限
  能妥協的 → 非核心信（衣服）
  不能妥協的 → 核心信候選

IdentityComposition ∧ DeathCriterion 的關係：
  單用 IdentityComposition → 混入重要但非核心的傾向
  單用 DeathCriterion → 混入情緒性反應
  兩者同時 → 雜訊排乾淨

核心信是二元的——在或不在。操作方法：只看結構，不看名稱

### 英雄

Hero ≡ 在核心信與現實出現落差時，試圖對齊的 Individual
  英雄的解法 = 穿衣服（給核心信套一層與現實相容的外殼）
  從 SFD.ExpressionGap 推出：衣服越假 → ExpressionGap 越大 → 需要越多封閉來維持

### 衣服

Garment ≡ 核心信的具體實現形式（不分敘事、制度、組織）
  判準：能被替換而核心信不死的 → Garment
  Garment 由一組 SFD.SignalContainer 組成
  功能：讓核心信在特定時代的 Signal 環境中降低 SFD.Filtering 的 SFD.Resistance，維持 SFD.PositiveFeedbackLoop 運作

### 功能性死亡

FunctionalDeath ≡ SFD.PositiveFeedbackLoop 的永久斷裂
  核心信 ¬ 因 SFD.Instability 自然衰減至死
  SFD.Instability → 殘差累積 ∧ 容器老化，¬ 信的消失
  核心信死亡的充分條件（二選一）：
    被另一個信替代（循環替代）
    ∨ 承載信的 Individual 被物理消滅

### 消音

Silencing ≡ SFD.PositiveFeedbackLoop ∧ SFD.KnowledgeBarrier 在制度層面的可觀察效果
  ¬ 獨立機制
  當 PositiveFeedbackLoop 通過控制 Signal 環境塑造所有 Individual 的 SemanticFlow 方向時 → 替代性輸出從源頭不被產出
  ¬ 壓制替代方案，替代方案從未出現
  識別標準：缺席有指紋（可追溯的 Power 行動維持缺席）∧ 想不到沒有

## 信模

來源：論文 §1.2

從 17 個反覆模式 + 12 個前提條件，經 CoreXinCriterion 篩選、合併，得三個組件：

### 核心信

CoreXin ≡ 天下有且只有一個中心
  世界的正確狀態是一個中心統攝一切
  ¬ 政策偏好，是 Xin 深層的預設值
  CoreXinCriterion 驗證：
    IdentityComposition — 歐洲、日本、印度無此預設
    DeathCriterion — 歷史上每次此結構受挑戰，文明付出最大代價恢復
  覆蓋模式 1, 2, 3, 4, 5, 7, 13, 15

### 動力機制

PowerMechanism ≡ 位置無主，誰贏歸誰
  唯一中心不屬於任何人、血統、神
  合法性來自勝利本身
  CoreXinCriterion 驗證：
    IdentityComposition — 歐洲君權神授、日本萬世一系非此結構
    DeathCriterion — 天命可轉讓性從未被放棄
  覆蓋模式 8, 10

### 再生產機制

ReproductionMechanism ≡ 同構組織
  一切內部複製了 CoreXin 結構的微觀組織形式
  家族是歷史上最耐久載體，¬ 唯一載體
  CoreXinCriterion 驗證：
    IdentityComposition — 中國家族同構程度在文明比較中獨特
    DeathCriterion — ReproductionMechanism 被徹底摧毀 → 核心信無法存續
  覆蓋模式 14

排序：CoreXin > PowerMechanism > ReproductionMechanism

## 蓄壓模型

來源：論文 §3.1

AccumulationModel ← SFD.Instability + 信模
  理論基礎：SFD.Instability（從 P1 + P2 + P4 推出）
  蓄壓模型是 Instability 在中國文明的信模條件下的特殊模式

六階段循環：

Accumulation（常態/蓄壓）：
  CoreXin 全面運作，SFD.PositiveFeedbackLoop 穩定
  表面穩定，底層蓄壓
  兩條線同時鋸：
    容器老化 ← Instability 內部來源
    衣服漂移 ← Instability 外部來源

SawThrough（鋸斷）：
  PowerMechanism 反覆衝擊容器
  大多數衝擊失敗，每次消耗容器資源
  最後成功的衝擊 ¬ 因為特別強，因為容器已被鋸至一層皮

Rupture（斷裂）：
  容器碎裂
  PowerMechanism 全面釋放
  CoreXin 沒斷
  ReproductionMechanism 在基層持續運作

Warring（混戰）：
  所有人爭奪唯一位置
  每個競爭者建小中央集權 ← ReproductionMechanism 微觀運作
  混戰期長度 ∝ 找 Garment 的難度

HeroPhase（英雄）：
  從混戰中勝出
  功能：找到 Garment，讓 CoreXin 在當下現實中存續
  合法性 ← PowerMechanism（勝利）
  Hero 缺位 → 長期混亂

Reconstruction（重建）：
  Hero 穿好 Garment
  ReproductionMechanism 接入新 SignalContainer
  PowerMechanism 被壓回沉睡
  PositiveFeedbackLoop 重啟 → Instability 重新累積

循環標註：Accumulation → SawThrough → Rupture → Warring → HeroPhase → Reconstruction → Accumulation
退出條件：FunctionalDeath（核心信被替代或承載者被消滅）

## 結構性推論

來源：論文 §3.3

以下推論從信模邏輯推出，不預測時間，只預測方向與約束

Inference₁: 改革路徑不存在
  CoreXin 要求唯一中心 ∧ 政治改革 = 限制中心 ∨ 製造獨立節點
  → 結構性矛盾 → KnowledgeBarrier 對此方向 Resistance 最大化

Inference₂: Garment 庫存不可逆趨近於零
  歷史可用 Garment 逐一失效 ← Instability 外部來源（不可逆）
  → 未來不會出現能長期包裝 CoreXin 的新 Garment

Inference₃: 經濟表現作為合法性替代品具有結構性脆弱
  當下幾乎只靠經濟維持合法性（歷史無先例）
  → 經濟長期失速 → 合法性缺乏備用支撐

Inference₄: 共和制或聯邦制在 CoreXin 存活條件下無法穩定運作
  CoreXin + ReproductionMechanism 存活 → FunctionalConvergence 驅動輸出趨近同一結構
  → 重建大概率仍為單一中心

Inference₅: CoreXin 死亡需要替代信，¬ 衝擊
  外部壓力可打碎 SignalContainer，¬ 殺死 PositiveFeedbackLoop
  → 需要替代循環

Inference₆: 數位封閉成本持續上升且不可逆
  維持封閉所需資源佔比持續擴大 ← Instability 外部來源在數位時代的表現
  → 封閉構成容器老化的加速因素

每條推論可由未來事實驗證或推翻 → 信模需對應結構性修正

## 可證偽條件

來源：論文 §0.7

能推翻 CoreXin：
  (1) 中國歷史出現穩定多中心政治結構（非外力維持，持續超過一代人，不以統一為預設目標）
  (2) 某次斷裂後重建為非單一中心結構（聯邦制、城邦制等），且為新常態非過渡態

能推翻 PowerMechanism：
  (3) 出現不可轉讓的合法性來源——超越性權威成功、持久地限制「誰贏歸誰」

能推翻 ReproductionMechanism：
  (4) 某次斷裂後所有同構組織被徹底摧毀，但 CoreXin 仍存活並驅動重建

能質疑框架區分力：
  (5) 對另一文明用同樣方法跑出同構信模
