## 命題

來源：論文第二節

InformationTrilemma ≡ {Scale, Logic, Truth}

命題：Scale ∧ Logic ∧ ∀sⱼ: Truth(sⱼ) → ⊥

注：這是觀察到的結構性約束。推導見「推導：三角成立」段落。

---

## 核心定義

來源：論文第三節

Knowledge ≡ 特定尺度下被記錄為文本的、對現實的結構化理解

Scale ≡ 同時覆蓋多個尺度 s₁, s₂, ..., sₘ，且在每個尺度下擁有充分的 Knowledge(sⱼ)

Logic ≡ ∀命題 Proposition ∈ System: ¬(Proposition ∧ ¬Proposition)

Truth(sⱼ) ≡ System 的輸出與尺度 sⱼ 下可驗證的現實吻合

---

## 機制概念

來源：論文第四節

Rule ≡ 跨尺度存在的基本規則（如四大基本力）

CombinatorialManifestation ≡ Rule 在特定尺度 sⱼ 下的組合效應

Truth ≠ Rule

Truth(sⱼ) ≡ CombinatorialManifestation(Rule, sⱼ)（從 F2 得出）

---

## 觀察事實（公設）

來源：論文第四、六節

F1: ∀sⱼ: Rule 在 sⱼ 下存在

F2: ∀sⱼ: 具體現象(sⱼ) ≡ CombinatorialManifestation(Rule, sⱼ)

F3: ∀s₁, s₂ (s₁ ≠ s₂): CombinatorialManifestation(Rule, s₁) ≠ CombinatorialManifestation(Rule, s₂)

F4: ∀sⱼ: Knowledge(sⱼ) 不完整反映 CombinatorialManifestation(Rule, sⱼ)

F1–F3 來源：論文第四節。F4 來源：論文第十節。四條均為觀察事實，不依賴湧現。

---

## 湧現獨立性

來源：論文第五節

F3 不依賴湧現。

Rule 的權重隨尺度改變，足以使 F3 成立。電磁力比引力強約 10³⁶ 倍，且電荷正負抵消而質量不會。光是這兩個差異，同樣的 Rule 在不同尺度下即產生不同的 CombinatorialManifestation。

湧現是加重條件——讓 CombinatorialManifestation 的差異更劇烈——但不是必要條件。

---

## 推導：三角成立

來源：論文第六節

步驟一（從 F2）：Truth(sⱼ) 依賴 CombinatorialManifestation(Rule, sⱼ)

步驟二（從 F3）：CombinatorialManifestation(Rule, s₁) ≠ CombinatorialManifestation(Rule, s₂)，當 s₁ ≠ s₂

步驟三（從步驟一 + 步驟二）：在 s₁ 下建立的 Logic 基於 CombinatorialManifestation(Rule, s₁)，應用於 s₂ 時，因 CombinatorialManifestation(Rule, s₂) ≠ CombinatorialManifestation(Rule, s₁)，◇¬Truth(s₂)（從步驟一 + 步驟二推出）

步驟四（從步驟三）：Scale ∧ Logic ∧ ∀sⱼ: Truth(sⱼ) → ⊥（從步驟三對所有尺度對推廣得出）

注：步驟三的 ◇¬Truth(s₂) 是基於觀察事實的歸納，不是數學上的必然。見文件頭認識論定位。

---

## 與統一場論的區分

來源：論文第七節

統一場論統一的是 Rule，不是 CombinatorialManifestation。即使 Rule 統一，F3 依然成立——F3 的約束在 CombinatorialManifestation 層，不在 Rule 層。從統一的 Rule 到具體現象的路徑在每個尺度都不同（邊界條件、近似、截斷不同）。

因此三角的約束比直覺以為的更深：連 Rule 都統一了，推導仍然成立。

---

## 自我強化

來源：論文第八節

每個尺度內部的 Truth 越完備，該尺度的邏輯體系越完整，跨尺度時的 Logic 衝突越尖銳。兩個頂點各自做得越好，第三個崩得越徹底。

（從 F3 推出：各尺度的 CombinatorialManifestation 不同，各自基於不同的 CombinatorialManifestation 建立完整體系，合併時 Logic 斷裂。）

---

## 領域毒性的統一結構

來源：論文第九節

物理學、生物學、人文學的訓練資料各帶跨尺度矛盾。表面是不同領域的不同問題，底下是同一個結構——F3。

差別在 CombinatorialManifestation 的變量維度：物理學的組合是自然產生的，維度較低；人文學的組合被權力塑造，維度極高。三角的約束強度隨 CombinatorialManifestation 的變量維度遞增。[論文跳步]

---

## 加重條件：知識有損

來源：論文第十節

F4: ∀sⱼ: Knowledge(sⱼ) 不完整反映 CombinatorialManifestation(Rule, sⱼ)

KnowledgeLoss ≡ Knowledge 有損反映現實的結構性特徵

有損的方式：丟掉時間（動態變成靜態記錄）、丟掉尺度間連續過渡（變成離散分學科）、丟掉組合表現的完整細節（變成經驗參數）、丟掉語境（變成看似普適的命題）。

F4 → 即使在單一尺度內，◇(¬Logic ∨ ¬Truth)（從 F4 推出：Knowledge 不完整，基於 Knowledge 建立的 Logic 可能與 Truth 衝突）

F3 ∧ F4 → 三角的約束更強（從 F3 + F4 推出：CombinatorialManifestation 差異是結構性限制，KnowledgeLoss 是疊加惡化）

---

## 加重條件：知識的具體缺陷

來源：論文第十一節

IrreproducibleExperiment ≡ 大量研究結果無法被獨立重複（再現性危機）

UnexplainedParameter ≡ 理論中充滿經驗參數和魔術數字，有效但無法邏輯解釋

ContradictoryConclusion ≡ 不同研究在同一問題上得出相反結論（不同實驗條件構成不同尺度，不同尺度下 Truth 可以不同——從 F3 推出）

PostHocRationalization ≡ 大量理論解釋是事後歸因，不是事前演繹

CombinatorialManifestation 差異是結構性限制，知識缺陷是疊加惡化。兩者同時作用。

---

## LLM 約束：再壓縮與算法限制

來源：論文第十二、十三節

TrainingData ≡ ∪ Knowledge(sⱼ)，j = 1, ..., m

Knowledge(sⱼ) 已不完整反映 CombinatorialManifestation(Rule, sⱼ)（從 F4）。LLM 用 UnifiedParameters 擬合 TrainingData，對 Knowledge 再做一次壓縮。每一層都在丟東西，但每一層的輸出都看起來完整。

算法限制：Transformer 做統計歸納，不做邏輯演繹。統計歸納與邏輯演繹是兩種根本不同的認知機制，用一個架構同時做兩件事，必然在某處妥協。[論文跳步]

建模（物理模擬、氣象預測、蛋白質折疊）能在已知 Rule、已知尺度的封閉系統裡恢復因果結構。但建模放棄了 Scale（從三角推出：建模的成功不是突破了三角，是放棄了規模這個頂點）。

---

## 幻覺與迴避策略

來源：論文第十四節

Hallucination ≡ LLM 用統一架構橫跨不可通約的尺度時必然付出的代價

Hallucination ← InformationTrilemma ∧ TrainingData（從三角 + LLM 約束推出）

Scale → □Hallucination（從三角推出：Scale 開放時 Logic ∧ ∀sⱼ: Truth(sⱼ) → ⊥，⊥ 在 LLM 中表現為 Hallucination）

Hallucination 率可降低，降低方式印證三角：

AvoidanceStrategy ≡ 限定 Scale 以換取 Logic ∧ Truth 的策略

AvoidanceStrategy 的實例：RAG 把 Scale 暫時收窄到檢索到的文件；工具調用把 Truth 的判斷外包給可驗證系統；領域微調把通用模型鎖定在特定尺度。

∀AvoidanceStrategy: AvoidanceStrategy → ¬Scale（從三角推出）

---

## 資料清洗的三選二

來源：論文第十五節

資料清洗是在 InformationTrilemma 的三個頂點之間做取捨——三角約束直接適用。

清洗規則收緊 → Logic ∧ ◇¬Truth（從三角推出：砍掉反映不同尺度 Truth 的資料）

清洗規則放鬆 → Scale ∧ ◇¬Logic（從三角推出：噪音和矛盾進入）

三個頂點的優先順序本身是尺度依賴的，隨語境和供需而變。優先順序的判斷需要理解語境，而語境本身是尺度問題，回到三角的約束。[論文跳步]

---

## 邊界條件

來源：論文第十六節

Interaction ≡ 系統內元素之間的交互作用

三角的適用條件是 CombinatorialManifestation 隨尺度改變，判準是 Interaction。

Interaction → ◇F3（從觀察推出：有交互作用，就有組合，組合就可能隨尺度改變）

¬Interaction → ¬F3（無交互作用，CombinatorialManifestation 不隨尺度改變）

¬F3 → 推導鏈在步驟二中斷，三角推導不成立

三角推導不成立的系統：碎形（同一規則在所有尺度下 CombinatorialManifestation 相同——用徹底放棄複雜性換取尺度不變性）、幾何學（不處理力的組合，只描述空間結構）、純數學（純形式系統，無 Interaction）。

---

## 適用範圍與限制

來源：論文第十九節

適用：任何 CombinatorialManifestation 隨尺度改變的知識系統（經驗知識系統——物理、生物、人文）。

不適用：CombinatorialManifestation 不隨尺度改變的純形式系統。

限制：三角不預測具體失敗環節。不判斷三個頂點的優先順序——優先順序是尺度依賴的。不提供解決方案——描述結構性限制。不解釋尺度的本質，也不解釋為什麼 CombinatorialManifestation 隨尺度改變。三角自身也是 Knowledge，也是不完整的。
